## ğŸ› **Troubleshooting**

### If the server doesn't start:
1. Make sure you're in the correct directory: `D:\Foundect`
2. Check if dependencies are installed: `npm install`
3. Use Command Prompt instead of PowerShell (Windows issue)
4. Check if port 3000 is available

### If you see errors:
1. Clear the `.next` folder: `rmdir /s /q .next` (Windows CMD)
2. Restart the dev server: `npm run dev`
3. Check the browser console for client-side errors

### If styles don't load:
1. Make sure `styles/glass.css` exists
2. Check `app/globals.css` imports the glass styles
3. Restart the dev server
